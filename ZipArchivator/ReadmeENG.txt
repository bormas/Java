Task name "Archivator"
Types of command line input:
1) To archivate file
name of a new archive + file or directory to archivate (+ file or directory to archivate)
2) To extract from archive
name of existing archive
	  or
name of existing archive + new directory name where we want to extract
3) To copy files to existing archive
name of existing archive + file or directory to copy (+ file or directory to copy)
